# E-commerce SyntaxWear

Este é um projeto de e-commerce para a loja SyntaxWear, especializada em roupas e acessórios de tecnologia.

## Funcionalidades

- Página inicial com produtos em destaque
- Catálogo de produtos
- Carrinho de compras
- Sistema de checkout

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Executar

Abra o arquivo `index.html` em um navegador web.
